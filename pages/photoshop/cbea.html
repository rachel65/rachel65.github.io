CBE Logo Submission

Created: March 6, 2015

Created in the course Web Design 2, this is a possible logo for the Colorado Business Educators. This group previously had no logo that could be used for professional purposes, and presented here is a possible image for them to use.

Photoshop techniques used include:

Pen tool
Used to outline an existing Colorado flag to make the current background
Selection tool (load selection)
An area of the red circle creating the Colorado flag was selected and erased using this tool
