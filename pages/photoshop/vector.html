Mitt Romney Vector Art

Created: March 9, 2015

This vector art rendition of politician Mitt Romney was created in the course Web Design 2. An original image of Romney (which can be found here) was selected and used as a reference to create the portrait as well as the basis for the flag.

Photoshop techniques used include:

Pen tool
Used to outline sections of the original reference image to create the portrait and flag
Posterize adjustment
Created an oversaturated, black and white version of the reference image for more obvious color variations
