Northern Colorado County Image Map

Created: February 25, 2015

Created in the course Web Design 2, this image map presents a map of Northern Colordao (which can be found here). Each county has been outlined and linked to open the corresponding county's website. It was made using Adobe Dreamweaver's image map function.

The code first identifies the Colorado image as the 'Map'. Polygons are then defined around areas which are determined by coordinates on the image. The target of each of them is then the county's website.
